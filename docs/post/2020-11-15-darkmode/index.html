Adding dark mode

  November 15, 2020  |  2 minutes

The main elements to add a “dark mode” functionnality:

  • a button to change the mode dark/light
  • a javascript function to apply change (ie adding or removing a “dark” class)
  • using localstorage to use the setting in all pages and for next visit
  • adding CSS for this dark mode

This work is based on the post https://yonkov.github.io/post/add-dark-mode-toggle-to-hugo/.

HTML template

We set the dark mode as the default ‘state’ using:

<body class="dark-mode">

Javascript

This work is based on the article. However:

  • dark is set as the default ‘state’
  • there are 3 states in the localStorage for darkMode: null (-> not yet set -> dark mode), true (-> darkMode), false (-> light mode)

The Javascript code added to main.js is:

    /* DARK MODE */
    function darkMode(){
        var body = document.body;
        var switcher = document.getElementsByClassName('js-toggle')[0];

        if(localStorage.getItem('darkMode')===null){
            // set dark as default
            console.log("set dark as default");
            setTimeout(function() {
                localStorage.setItem('darkMode','true');
            }, 100);
        }
        else{
            var darkMode=localStorage.getItem('darkMode');
            if(darkMode=='true'){
                // do nothing because we are in dark mode
            }
            else {
                switcher.classList.toggle('js-toggle--checked');
                body.classList.remove('dark-mode');
            }
        }

        //Click on dark mode icon. Add dark mode classes and wrappers. Store user preference through sessions
        switcher.addEventListener("click", function() {
            this.classList.toggle('js-toggle--checked');
            this.classList.add('js-toggle--focus');
            //If light mode is selected
            if (this.classList.contains('js-toggle--checked')) {
                body.classList.remove('dark-mode');
                //Save user preference in storage
                setTimeout(function() {
                    localStorage.setItem('darkMode','false');
                }, 100);
            } else {
                body.classList.add('dark-mode');
                setTimeout(function() {
                    localStorage.setItem('darkMode','true');
                }, 100);
            }
        })

    }
